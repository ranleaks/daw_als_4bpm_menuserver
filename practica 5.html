La Historia de HTML

HTML por sus siglas en ingles de HyperText Markup Languaje ('lenguaje de marcas de hipertexto')
hace referencia al lenguaje de marcado para la elaboracion de paginas web.
Es un estanadr que sirve de referencia del software que conecta con la elaboracion
de paginas web en sus diferentes versiones, define una estructura basica
y un codigo (denominado codigo HTML) para la definicion de contenido
de una pagina web, como texto, imagenes, videos, juegos, entre otros.

  • Version 5.0 2012
